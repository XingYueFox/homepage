# 星月_Fox的个人主页

这是一个基于 [SRInternet 开发的 Home_Page](https://github.com/SRInternet/Home_Page/tree/master)的二次创作版本。

## 🔄 新增功能

- ✨ **设计风格**：在大体上与 Home_Page 设计相似
- 🪟 **自适应布局**：完美适配从手机到桌面的各种屏幕尺寸（再次
- 🎨 **动画效果**：在原有动画之上，我加入了涟漪交互动画
- 📃 **全新的顶部栏**：仿照 Apple 官网为 Homepage 加上全新的可交互顶部栏
- 🗑️ **去GitHub化**：这对更多人更友好（详见下
- 📈 **活动概览？**：被我换成了个人评估概览（这是计划的一部分
- 📝 **技术栈展示**：更换了配色和描边设计，使页面看上去更加清爽
- 🕒 **最近项目展示**：没那么多项目，被我换成了快速链接
- 🎮 **游戏标签**：拥有流畅的交互特效+浮动卡片效果的游戏板块

## 🛠️ 技术栈

- **后端**：Python 3.6+, Flask
- **前端**：HTML5, Tailwind CSS, JavaScript
- **数据可视化**：Chart.js
- **第三方集成**：GitHub API（可选）
- **部署选项**：Clouldflare Pages，GitHub Pages, Vercel, Netlify等

## ❔ 为什么会有这个
对于一些 Github 上啥都没有的人，Home_Page 完美诠释了何为一穷二白。并且，Home_Page 的模板本身较为基础，为此我想制作一个更好的 Home_Page 版本:Homepage 旨在为更多更多用户提供更低的入手门槛。<br>
现在，你不会再被这些数据束缚，你可以更方便的在这上面写上你的各种信息。<br>
